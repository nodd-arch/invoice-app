# Invoice & Receipt Generator

A professional invoice and receipt generator built with Next.js + TypeScript.

## Features
- Toggle between Invoice and Receipt mode
- Live preview as you type
- M-Pesa, Bank Transfer, Cheque, PayPal payment methods
- VAT handling (included / calculated)
- Draw-to-sign signature pad
- Download as PDF

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Vercel

### Option A — Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option B — GitHub + Vercel Dashboard
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import your GitHub repo
4. Framework: **Next.js** (auto-detected)
5. Click **Deploy** — done!

No environment variables needed.
